**8 World-First Engines | Never before adapted to commercial AI systems**

Orchestrated by: `services/NSILIntelligenceHub.ts` (master control for all 15 engines)

### CRE — Creative Synthesis Engine

**Theory:** Bisociation Theory (Arthur Koestler, *The Act of Creation*, 1964)

- 12 knowledge frames (marine biology, military strategy, game theory, etc.)
- Jaccard similarity and cosine distance between frame pairs
- Identifies bisociative connections where dissimilar frames share structure
- Generates novel strategies no single-domain expert would propose
- Novelty score, feasibility assessment, implementation pathway for each

**Implementation:** `services/autonomous/CreativeSynthesisEngine.ts` (~450 lines)

### CDT — Cross-Domain Transfer Engine

**Theory:** Structure Mapping Theory (Dedre Gentner, 1983)

**Eight source domain models:**
- Coral Reef Ecosystem
- Immune System
- Military Strategy
- Thermodynamics
- Neural Networks
- Lotka-Volterra Population Dynamics
- Urban Metabolism
- Game Theory

- 50+ entity-to-economic-concept mappings per domain
- Evaluates structural similarity, relational depth, pragmatic utility
- Analogies ranked by composite score
- Each includes mapped insights with strategic recommendations

**Implementation:** `services/autonomous/CrossDomainTransferEngine.ts` (~450 lines)

### AGL — Autonomous Goal Detection Engine

**Theory:** Goal Programming + HTN Decomposition + MCDA

- 8 detection rules scan for unaddressed objectives
- Tax incentive gaps, supply chain fragility, talent risks, regulatory arbitrage
- Goals decomposed into sub-goals via HTN
- Ranked using MCDA: 0.30×impact + 0.25×urgency + 0.20×feasibility + 0.25×EVOI
- Goals above threshold surfaced proactively

**Implementation:** `services/autonomous/AutonomousGoalEngine.ts` (~380 lines)

### ETH — Ethical Reasoning Engine

**Theory:** 7-Dimension Ethical Framework

**Acting as HARD GATE:**
- Ethics score below threshold → Recommendation BLOCKED (not flagged)

**7 Dimensions:**
- Utilitarian calculus
- Rawlsian Difference Principle (worst-off group benefit)
- Environmental impact
- Intergenerational equity (Stern rate r=1.4%)
- Transparency
- Proportionality
- Cultural Sensitivity

- Gini coefficient: distributional equality
- Stern formula: PV = FV × e^(-0.014 × 30)
- 7 auto-generated stakeholder groups
- Output: **Proceed / Proceed-with-conditions / Redesign / Reject**

**Implementation:** `services/autonomous/EthicalReasoningEngine.ts` (~350 lines)

### EVO — Self-Evolving Algorithm Engine

**Theory:** Online Gradient Descent + Thompson Sampling

- 21 weight parameters continuously tuned based on outcomes
- Learning rate: η_t = 0.05 / (1 + 0.001 × totalMutations)
- ε-greedy exploration with ε decaying from 0.3
- Immutable audit trail with rollback capability
- Mutations validated before permanent commitment

**Implementation:** `services/autonomous/SelfEvolvingAlgorithmEngine.ts` (~320 lines)

### ADA — Adaptive Learning Engine

**Theory:** Bayesian Normal-Normal Conjugate Updates + EWMA + Ebbinghaus

- 15 prior beliefs updated via Bayesian posterior updates
- EWMA tracking: α=0.1
- Ebbinghaus forgetting: R = e^(-t/S) where S = 24 × √(reinforcement count)
- Patterns that recur strengthen
- Patterns not reinforced decay

**Implementation:** `services/autonomous/AdaptiveLearningEngine.ts` (~330 lines)

### EMO — Emotional Intelligence Engine

**Theory:** Russell's Circumplex Model + Prospect Theory

- 12 emotions mapped onto valence/arousal
- Four stakeholder profiles: Investor, Regional Leader, Community, Regulator
- Prospect Theory decision-weighting: V(x) = x^0.88 (gains), V(x) = -2.25 × (-x)^0.88 (losses, λ ≈ 2.25)
- Probability weighting: Prelec function with γ = 0.61
- Derailment risk score: emotional factors overriding rational analysis

**Implementation:** `services/autonomous/EmotionalIntelligenceEngine.ts` (~350 lines)

### SIM — Scenario Simulation Engine

**Theory:** Monte Carlo + System Dynamics + Causal Loop Diagrams

- 12 scenario variables connected by 14 causal links
- Four non-linearity types: linear, quadratic, threshold, saturation
- Four feedback loops: Growth Spiral, Infrastructure-Investment, Talent Development, Inflation Brake
- 5,000 Monte Carlo iterations per run
- Box-Muller normal sampling, triangular distributions, lognormal
- Perturbations propagated through causal network

**Implementation:** `services/autonomous/ScenarioSimulationEngine.ts` (~380 lines)

**Output:** Echo connections, contradictions, system knowledge matches, "self-answer score"

### InvestmentLifecycleMapper

**Theory:** Product Lifecycle Theory (Vernon, 1966) + Kondratieff + Schumpeterian Creative Destruction

**Maps region's investment lifecycle position:**  
Emergence → Growth → Boom → Peak → Plateau → Decline → Dormancy → Reactivation

- Current phase: FDI trends, population, infrastructure age, govt priority
- Maps recyclable assets from previous boom periods
- Predicts phase transitions with catalysts/blockers
- Finds similar regions at different lifecycle stages
- Generates phase-specific reactivation playbooks

**Output:** Phase ID, recyclable asset inventory, phase transition prediction, structural twins, ordered reactivation playbook

**Implementation:** `services/reflexive/InvestmentLifecycleMapper.ts`

### RegionalMirroringEngine

**Theory:** Structural Similarity Analysis + Analogical Reasoning

**Finds region's structural twins** (NOT aspirational peers):
- Similar economic profiles, demographics, infrastructure, geography, institutions, sectors
- Compares across 6 dimensions with weighted similarity scoring
- Regions benchmark against "the next Singapore" instead of structural matches

**Output:** Top 5 structural mirrors with transferable strategies, aspiration gap analysis, cross-mirror strategy patterns

**Implementation:** `services/reflexive/RegionalMirroringEngine.ts`

### RegionalIdentityDecoder

**Theory:** Baudrillard's Simulacra + Porter's Regional Competitiveness + Narrative Theory

**Detects Simulacrum Phase** (loss of authentic competitive identity):

- Scans for 10 categories of generic investment promotion language
- Detects aspirational mimicry ("the next Singapore")
- Identifies hidden assets being undersold
- Measures gap between narrative strength and evidence strength
- Categorises identity loss from "none" to "complete"

**Output:** Authenticity score, simulacrum level, identity element analysis, hidden asset detection, identity gaps, positioning recommendations

**Implementation:** `services/reflexive/RegionalIdentityDecoder.ts`

### LatentAdvantageMiner

**Theory:** Hidden Asset Theory + Porter's Diamond

**Scans for casually mentioned global investment attractors:**

**Output:** Latent advantages with global precedents, "junk DNA ratio," blind spot questions, strategic value assessment, narrative gap analysis

**Implementation:** `services/reflexive/LatentAdvantageMiner.ts`

## ━━━ GLOSSARY ━━━

**Driver:** Factor that increases a score.

**Pressure point:** Factor that reduces a score or increases risk.

**Gating factor:** Something that blocks activation until resolved.

**P10/P50/P90:** Optimistic / Median / Conservative scenario bounds.

**Bisociation:** Creative connection between two unrelated frames of reference.

**Structural twin:** Region with similar economic/demographic/institutional profile (not aspirational peer).

**Simulacrum:** Loss of authentic identity; generic, interchangeable positioning.
